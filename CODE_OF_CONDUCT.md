# Code of Conduct / 行为准则

[English](#english) | [简体中文](#简体中文)

## English

This project follows a simple rule: keep discussion respectful, technical, and useful.

Expected behavior:

- Be clear and constructive when reporting issues or reviewing code.
- Do not post secrets, tokens, private chat IDs, personal data, or screenshots containing credentials.
- Respect that this is a local-first tool with security-sensitive integrations.

Unacceptable behavior:

- Harassment, insults, or personal attacks.
- Publishing another person's secrets, credentials, logs, local paths, or private data.
- Repeated off-topic or disruptive comments.

Maintainers may edit, hide, or remove content that violates these rules.

## 简体中文

本项目遵循一个简单原则：讨论应保持尊重、技术性和有实际帮助。

期望行为：

- 提 issue 或 review 代码时，保持清晰、具体、建设性。
- 不要发布密钥、token、私有 chat ID、个人数据，或包含凭证的截图。
- 尊重本项目本地优先且涉及敏感集成的安全边界。

不可接受行为：

- 骚扰、侮辱或人身攻击。
- 公开他人的密钥、凭证、日志、本地路径或隐私数据。
- 重复发布无关或干扰性内容。

维护者可以编辑、隐藏或删除违反这些规则的内容。

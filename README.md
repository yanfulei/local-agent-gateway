# Local Agent Gateway

[English](./README.en.md) | [简体中文](./README.zh-CN.md)

Local Agent Gateway is a local-first gateway for exposing desktop coding-agent sessions to chat platforms.

本项目是一个本地优先的智能体网关，用于把电脑上的本地编程智能体会话代理到飞书等即时通讯渠道。

The current MVP supports macOS, Codex CLI app-server/remote-control, a local Web UI, and Feishu/Lark long-connection bots.

当前 MVP 支持 macOS、Codex CLI app-server/remote-control、本地 Web UI，以及飞书长连接机器人。

## Quick Start

```bash
npm install
npm run build
npm start
```

Open http://127.0.0.1:3030.
